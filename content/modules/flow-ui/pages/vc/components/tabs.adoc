= tabs 标签页

<div class="jmix-ui-live-demo-container">

`tabs` 创建一组标签页，使用标签页可以更换视图中的内容。与 xref:vc/layouts/tabSheet.adoc[tabSheet] 不同，该组件不能包含标签内容。

* XML 元素：`tabs`

* Java 类：`Tabs`

== 标签页

独立的标签页使用内部的 `tab` 元素定义。

[TIP]
====
如需在 Jmix Studio 中添加 `tab`，在视图描述 XML 或 *Jmix UI* 层级面板中选择组件，然后点击 *Jmix UI* 组件面板的 xref:studio:view-designer.adoc#component-inspector-add-button[Add]*->Tab* 按钮。
====

==== flexGrow

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/flex-grow" class="mdn-docs-btn" target="_blank">MDN</a>

=== orientation

++++

定义标签页方向。支持下列值：
