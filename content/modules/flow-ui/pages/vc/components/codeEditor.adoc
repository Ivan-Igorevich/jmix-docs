`codeEditor` 是一个用于展示或输入源代码的组件。

`codeEditor` 是一个多行文本域，提供代码高亮显示、打印边距以及带有行号的侧边栏。

* XML 元素：`codeEditor`

* Java 类：`CodeEditor`

[[fontSize]]
=== fontSize

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/font-size" class="mdn-docs-btn" target="_blank">MDN</a>
</div>
++++

设置编辑器内源代码的字体大小。默认 `1rem`。

[[highlightActiveLine]]
=== highlightActiveLine

设置是否高亮当前行。默认启用。

[[mode]]
=== mode

设置语法高亮的模式。支持的模式由 `CodeEditorMode` 枚举定义。

默认语法高亮模式是 `CodeEditorMode.PLAIN_TEXT`。

[[printMarginColumn]]
=== printMarginColumn

设置代码宽度垂直线的位置。默认 80。

[[showGutter]]
=== showGutter

设置侧边栏的可见性。默认可见。

[[showLineNumbers]]
=== showLineNumbers

设置行号的可见性。默认可见。

[[showPrintMargin]]
=== showPrintMargin

设置代码宽度垂直线的可见性。默认可见。

[[theme]]
=== theme

设置 `codeEditor` 的可视化主题。支持的主题由 `CodeEditorTheme` 枚举定义。

默认的主题为 `CodeEditorTheme.TEXTMATE`。
