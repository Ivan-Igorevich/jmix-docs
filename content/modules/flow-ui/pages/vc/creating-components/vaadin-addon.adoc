= Интеграция дополнений Vaadin

В Jmix-приложениях можно использовать любые сторонние дополнения Vaadin, расположенные в https://vaadin.com/directory[Vaadin Directory^] или в любом другом репозитории. Пример ниже демонстрирует интеграцию дополнения https://vaadin.com/directory/component/pdf-viewer[Pdf Viewer^].
