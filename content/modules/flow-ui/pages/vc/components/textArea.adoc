`textArea` 是一个多行文本编辑控件。

* XML 元素：`textArea`

* Java 类：`JmixTextArea`

== 基本用法

`textArea` 的功能与 xref:vc/components/textField.adoc[textField] 基本相同。

如果未设置固定高度，`textArea` 会根据内容自动调整高度。默认高度和最小高度是两行文字。
