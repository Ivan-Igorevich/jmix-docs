= Superset

Superset 扩展组件可以在 Jmix 应用程序中集成 https://superset.apache.org/[Apache Superset^]。扩展组件中提供了 `SupersetDashboard` 组件和相关功能，可以在应用程序中集成 Apache Superset 中创建的仪表板。

在应用程序中集成仪表板时，只需配置 Superset 的 URL 和登录凭证，然后在视图中添加 `SupersetDashboard` 组件。
