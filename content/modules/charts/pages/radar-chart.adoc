= Радарная Диаграмма

В этом разделе вы:

* Создадите радарную диаграмму
* Добавите элементы `indicator` и `title`
* Привяжите данные через Data Container

[[setup]]
== Настройка Радарной Диаграммы

Вы интегрируете компонент радарной диаграммы, чтобы иллюстрировать метрики навыков сотрудников: `hardSkills`, `problemSolving`, `softSkills` и `teamwork`.

[[series]]
=== Настройка Серии Радарной Диаграммы

Давайте вставим компонент диаграммы в `user-list-view.xml`.

Сначала щелкните по `formLayout`, где будет расположена диаграмма.

Выберите *Add Component* на панели действий и выберите элемент *Chart*, чтобы вставить элемент `<charts:chart/>`. Настройте атрибуты `id`, `height` и `width` следующим образом.

[source,xml,indent=0]
----
include::example$charts-ex2/src/main/resources/com/company/onboarding/view/user/user-list-view.xml[tags=radarChartElement]
----

Затем добавьте серию радар в ваш график. Щелкните по элементу `chart` в панели иерархии *Jmix UI* или в дескрипторе экрана XML, затем нажмите кнопку *Add* на панели инспектора. Выберите *Series -> radar* из выпадающего списка.

[source,xml,indent=0]
----
include::example$charts-ex2/src/main/resources/com/company/onboarding/view/user/user-list-view.xml[tags=radarSeries]
----

== Добавление Свойств Диаграммы

[[indicators]]
=== Индикаторы

Радарная диаграмма с несколькими осями требует индикаторов в отдельном элементе `radar` под `chart`. Индикаторы представляют ось каждого метрического показателя.

Чтобы добавить индикаторы, вручную создайте элемент `<charts:radar/>`. Выберите элемент `radar` и выберите *Add -> Indicators -> Indicator*.

Определите каждый навык как индикатор с атрибутами `name` и `max`, чтобы представить шкалу следующим образом.

[source,xml,indent=0]
----
include::example$/charts-ex2/src/main/resources/com/company/onboarding/view/user/user-list-view.xml[tags=indicators]
----

[[data-binding]]
== Привязка Данных с Использованием Data Container

Для вашей радарной диаграммы вы можете использовать data container `emloyeesDc`, который вы настроили в разделе xref:charts:bar-chart.adoc#data-binding[столбчатой диаграммы].

Чтобы включить полученные данные в диаграмму, выберите компонент `chart` и вручную добавьте атрибут `dataSet`.

[source, xml]
----
<charts:dataSet/>
----

Внутри тега `dataSet` выберите контейнер `employeesDc` для вашего атрибута `source`. Радарной диаграмме нужны только `valueFields` как ее оси.

[source,xml,indent=0]
----
include::example$/charts-ex2/src/main/resources/com/company/onboarding/view/user/user-list-view.xml[tags=radarTitle]
----

По завершении этих шагов ваша конфигурация XML для радарной диаграммы будет следующей.

[source,xml,indent=0]
----
include::example$/charts-ex2/src/main/resources/com/company/onboarding/view/user/user-list-view.xml[tags=radarChart]
----
