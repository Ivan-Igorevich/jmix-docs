= 表格导出操作

该扩展组件为 Web 应用程序中的表格组件提供了一个非常有用的功能。支持用户从表格中下载数据至不同格式的文件，例如 Excel 或 JSON。

扩展组件提供一些 xref:grid-export:actions.adoc[操作]，用于触发导出功能。

这些操作可以在 xref:flow-ui:vc/components/dataGrid.adoc[] 和 xref:flow-ui:vc/components/treeDataGrid.adoc[] 组件内使用。

[[installation]]
== 安装

按照 xref:ROOT:add-ons.adoc#installation[扩展组件] 章节的说明通过 Jmix 市场进行自动安装。

[[usage]]
== 用法

扩展组件提供了两种类型的操作：xref:grid-export:actions.adoc#excel-export-action[grdexp_excelExport] 和 xref:grid-export:actions.adoc#json-export-action[grdexp_jsonExport]。两种操作都是 xref:flow-ui:actions/list-actions.adoc[列表组件操作]，因此可以添加至 xref:flow-ui:vc/components/dataGrid.adoc[dataGrid] 和 xref:flow-ui:vc/components/treeDataGrid.adoc[treeDataGrid]。

项目中添加了该扩展组件后，Jmix Studio 中可以通过 xref:studio:view-designer.adoc#add-action[addition] 添加操作。

image::export-action-studio.png[align="center",width="492"]

示例：

[source,xml,indent=0]
----
include::example$/gridexport-ex1/src/main/resources/com/company/demo/view/user/user-list-view.xml[tags=buttonsPanel;export;dataGrid;actions]
----

如需启用表格的多选，请设置 `dataGrid` 组件的 `selectionMode="MULTI"`。
