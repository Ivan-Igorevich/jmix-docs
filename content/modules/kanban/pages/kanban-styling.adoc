= Styling

In this chapter, you will create:

* Actions to control the visibility of columns.
* Visual styles of a component using properties.
* Renderers for column footers.

[[adding-column-actions-buttons]]
== Adding Column Actions Buttons

++++
<div class="jmix-ui-live-demo-container">
    <a href="http://demo.jmix.io/ui-samples/sample/kanban-column-actions" class="live-demo-btn" target="_blank">LIVE DEMO</a>

The Kanban board component supports dynamic changes to display and visibility settings of columns. To enable this feature, you need to set the xref:kanban-component.adoc#columnActionsEnabled[columnActionsEnabled] attribute. This attribute adds a column visibility control button to the header. To be able to collapse a column, you need to set the xref:kanban-component.adoc#collapsible[collapsible] attribute.

Locate `kanban-task-list-view.xml` in the *Jmix* tool window and double-click it. Choose `kanban` within the *Jmix UI* hierarchy panel or in the XML descriptor. Next check `collapsible` and `columnActionsEnabled` boxes:

image::styling-attributes.png[align="center",width="700"]

Press *Ctrl/Cmd+S* and switch to the running application. Open the `Kanban Tasks` view. Column visibility controls are new available.

[TIP]
====
To save the state of the columns after closing the view, you can use the xref:flow-ui:facets/settings.adoc[Settings Facet].
====

There are many Kanban attributes that change the visual appearance of the component. You need to add the following attributes:

* `columnSummaryEnabled`, `columnFooterVisible` – to display the summary.
* `columnColorEntireSurfaceEnabled` – to highlight the background of the columns with a contrasting color.
* `dropPlaceholderAllowed` – to add a placeholder when moving task cards.
* `color` for columns, `applyColumnColorToTasks` – for special highlighting of each column.

All these attributes can be added using the *Jmix UI* component inspector panel or using XML markup.

[[renderer-for-column]]
== Renderer for Column

The `kanban` component supports custom JavaScript renderers to style the display the task cards and columns. See xref:kanban-component.adoc#renderers[renderers] section.

To additionally highlight a column that has too many tasks, you need to define a renderer for the column footer.

To do this, select `kanban` in the *Jmix UI* hierarchy panel or in the XML descriptor, then click the *Add* button in the component inspector panel. From the drop-down list, choose *ColumnFooterRendererFunction*:

image::add-column-footer-render.png[align="center",width="1004"]

`ColumnFooterRendererFunction` is defined by a callback function which can be used for customizing column footer rendering. The callback has three arguments: column footer HTML element, column data and column data field.
