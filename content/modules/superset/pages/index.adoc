= Superset

Дополнение Superset предназначено для интеграции приложений Jmix с https://superset.apache.org/[Apache Superset^]. Оно предоставляет компонент `SupersetDashboard` и связанные с ним функции, которые позволяют встраивать дэшборды, созданные в Apache Superset, в экраны приложений.

Чтобы отобразить дэшборд в вашем приложении, вам нужно только настроить URL и учетные данные Superset, а затем добавить компонент `SupersetDashboard` к экрану.
