= drawerToggle 侧边栏开关

控制应用程序菜单显示的开关。

* XML 元素：`drawerToggle`

* Java 类：`DrawerToggle`

== 基本用法

`drawerToggle` 组件是一个开关按钮，用于控制包含 xref:menu-config.adoc[主菜单] 的 `drawerLayout` 元素的打开和关闭状态。

image::visual-components/components/drawer-toggle-basic.png[align="center", width="450"]

为了使用户在整个应用程序中都能访问主菜单，`drawerToggle` 一般放置在主应用程序视图的 `navigationBar` 组件中。

[[themeNames]]
== 样式版本

通过 xref:vc/common-attributes.adoc#themeNames[themeNames] 属性自定义组件的颜色和大小。

=== 配色

有四种颜色可选：default、`success`、`error` 以及 `contrast`。将配色与 `primary` 主题一起使用可以更加显著地显示。

=== 尺寸

有三种尺寸可选：`small`、default、`large`。

[[ClickEvent]]
=== ClickEvent

`com.vaadin.flow.component.ClickEvent` 当用户点击组件时发送。处理方法需要指定下列三种类型的点击之一。

* click - 点击时触发

* singleClick - 在一定的延时后触发，以确保不是双击。

* doubleClick - 当双击组件时触发。
