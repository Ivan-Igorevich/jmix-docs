= 样式

使用 `Style` 对象可以自定义地图 xref:maps:features-geometries.adoc[要素] 的颜色、大小、字体和其他可视化属性。

样式也可以在 xref:maps:layers-sources.adoc[图层] 级别应用，从而为图层中的所有要素定义一致的外观。

== PointFeature 样式

如需更改 xref:maps:features-geometries.adoc#point-feature[PointFeature] 的样式，可以定义一个 `PointStyle` 对象，设置所需的外观属性（如 `image`、`fill`、`stroke` 等），然后将该样式应用于 `PointFeature`。

[source,java,indent=0]
----
include::example$/maps-ex2/src/main/java/com/company/mapssample/view/style/StyleView.java[tags=point]
----

image::style-point.png[align="center", width="480"]

为 `PointFeature` 设置文本而非图像，可以直接在地图上的指定坐标处显示文本标注，从而为显示的数据提供清晰的上下文。

`PointFeature` 使用文本时，可以通过调整 `font`、`fill`、`textAlign` 等属性来自定义文本的外观。
