Дополнение Jmix Charts интегрирует https://echarts.apache.org[Apache ECharts^], библиотеку визуализации на JavaScript с открытым исходным кодом, в Jmix UI. Эта интеграция позволяет разработчикам использовать ECharts для создания широкого спектра графиков.

[[installation]]
