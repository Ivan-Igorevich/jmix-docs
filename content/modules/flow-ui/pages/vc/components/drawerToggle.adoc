= drawerToggle 侧边栏开关

`drawerToggle` 是页面左侧可折叠 `drawerLayout` 的控制器。左侧边栏包含应用程序的主菜单。

* XML 元素：`drawerToggle`

* Java 类：`DrawerToggle`

== 基本用法

`drawerToggle` 主要放置于 xref:views.adoc#standard-main-view[应用程序主视图] 内，与 `navigationBar` 和 `drawerLayout` 一起使用，实现应用程序内部导航的一种通用模式。

在默认的应用程序模板中，`drawerToggle` 位于导航栏的左上角。

image::visual-components/components/drawer-toggle-basic.png[align="center", width="450"]

用户可以通过这个组件在应用程序的任何页面访问主菜单，或者可以折叠主菜单提供更多空间。
