`comboButton` - это кнопка с выпадающим списком элементов. `comboButton` используется, чтобы предоставить набор вариантов xref:actions/declarative-actions.adoc[действий].

Этот компонент очень похож на xref:vc/components/dropdownButton.adoc[dropdownButton].

Основное отличие заключается в том, что к корневой кнопке можно добавить действие.
