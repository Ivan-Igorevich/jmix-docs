++++

`listBox` 支持用户从可滚动的列表选择单一值。

* XML 元素：`listBox`

* Java 类：`JmixListBox`

== 基本用法

`listbox` 支持使用枚举或实体集合作为选项列表。

用 xref:vc/components/comboBox.adoc#itemsEnum[itemsEnum] 属性直接指定枚举类的名称：

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/listbox/list-box-view.xml[tags=itemsEnum]
----

如果是实体集合，则可以先定义一个集合数据容器，然后设置 xref:vc/components/entityComboBox.adoc#itemsContainer[itemsContainer] 属性：

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/listbox/list-box-view.xml[tags=data;citiesDc;layout;itemsContainer]
----

<1> `City` 实体的 `CollectionContainer`。
<2> 设置 `citiesDc` 作为组件的选项容器。
