`checkbox` 是一个拥有两个状态的组件：选中或者未选中。

* XML 元素：`checkbox`

* Java 类：`JmixCheckbox`

== 数据绑定

该组件支持绑定至 xref:data/data-containers.adoc[data container] 中的实体属性。

如需将 `checkbox` 绑定至实体属性：

. 使用 xref:flow-ui:vc/common-attributes.adoc#dataContainer[dataContainer] 属性指定数据容器的名称。
. 使用 xref:flow-ui:vc/common-attributes.adoc#property[property] 属性指定实体属性的名称。

实体属性必须为 `Boolean` 类型。

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/checkbox/check-box-view.xml[tags=data;layout;data-binding]
----

设置 `checkbox` 的辅助功能标签。

[[indeterminate]]
=== indeterminate

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://vaadin.com/docs/latest/components/checkbox/#indeterminate" class="vaadin-docs-btn" target="_blank">Vaadin</a>
</div>
++++

设置 `checkbox` 的不确定状态。

NOTE: 根据 HTML5 标准，这个状态只是影响组件的外观，并不会影响组件的值。

[[value]]
=== value

定义 `checkbox` 的值，可以是 `true` 或 `false`。
