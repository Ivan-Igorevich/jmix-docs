= checkboxGroup 复选框组

++++
<div class="jmix-ui-live-demo-container">

`checkBoxGroup` 支持用户使用一组复选框选择多个值。

* XML 元素：`checkBoxGroup`

* Java 类：`JmixCheckboxGroup`

== 基本用法

`checkBoxGroup` 最简单的用法是用于选择枚举值。

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/checkboxgroup/checkboxgroup-view.xml[tags=basics]
----

image::visual-components/components/check-box-group-basics.png[align="center", width="156"]

== 数据绑定

该组件支持绑定至 xref:data/data-containers.adoc[data container] 中的实体属性。

如需将 `checkBoxGroup` 绑定至实体属性：

. 使用 xref:flow-ui:vc/common-attributes.adoc#dataContainer[dataContainer] 属性指定数据容器的名称。
. 使用 xref:flow-ui:vc/common-attributes.adoc#property[property] 属性指定实体属性的名称。

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/checkboxgroup/checkboxgroup-view.xml[tags=data;userDc;hobbiesDc;layout;data-bind]
----

示例中，`checkBoxGroup` 会展示 `Hobby` 实体的实例名，`getTypedValue()` 方法返回选择实例的 `Collection`.

=== themeNames

使用 xref:vc/common-attributes.adoc#themeNames[themeNames] 属性定义选项的布局方式。

默认情况下，选项水平排列。
