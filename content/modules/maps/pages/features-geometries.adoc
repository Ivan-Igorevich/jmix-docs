= 地图要素和几何图形

`GeoMap` 下列地图要素：

* <<marker-feature,MarkerFeature>>
* <<point-feature,PointFeature>>
* <<line-string-feature,LineStringFeature>>
* <<polygon-feature,PolygonFeature>>

[[marker-feature]]
== MarkerFeature

标记要素表示使用预定义 <<point-geometry,PointGeometry>> 的一个标记元素。在地图组件中显示为一个标记图标。

[[line-string-feature]]
== LineStringFeature

`LineStringFeature` 表示地图中的线性要素，一般是多段线。使用预定义的 <<line-string-geometry,LineStringGeometry>>。更多内容请参阅 https://openlayers.org/en/v8.1.0/apidoc/module-ol_Feature-Feature.html[要素文档^]。

[[polygon-feature]]
== PolygonFeature

`PolygonFeature` 表示地图中的封闭几何图形，定义一个区域或地区。这个要素用一组互相连接的点组成一个封闭的图形，一般是多边形。使用预定义的 <<line-string-geometry,LineStringGeometry>>。更多内容请参阅 https://openlayers.org/en/v8.1.0/apidoc/module-ol_Feature-Feature.html[要素文档^]。

[[point-geometry]]
== PointGeometry

`PointGeometry` 是一个包含 JTS 库中 `org.locationtech.jts.geom.Point` 的对象。

[[line-string-geometry]]
== LineStringGeometry

`LineStringGeometry` 是一个包含 JTS 库中 `org.locationtech.jts.geom.LineString` 的对象。

[[polygon-geometry]]
== PolygonGeometry

`PolygonGeometry` 是一个包含 JTS 库中 `org.locationtech.jts.geom.Polygon` 的对象。
