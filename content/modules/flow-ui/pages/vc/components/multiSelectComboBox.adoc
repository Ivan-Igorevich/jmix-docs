= multiSelectComboBox 多选下拉框

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://demo.jmix.io/ui-samples/sample/multi-select-combo-box-simple" class="live-demo-btn" target="_blank">在线示例</a>

++++

`multiSelectComboBox` 支持用户从下拉列表选择一项或多项。

该组件与 xref:vc/components/comboBox.adoc[comboBox] 类似。

* XML 元素：`multiSelectComboBox`

* Java 类：`JmixMultiSelectComboBox`

== 基本用法

当用户点击控件时可以打开下拉列表。使用键盘的上下键或者在控件中输入内容也可以打开下拉列表。

image::visual-components/components/multi-select-combo-box-basics.png[align="center", width="255"]

如需创建关联至数据的 `multiSelectComboBox`，请使用 xref:vc/common-attributes.adoc#dataContainer[dataContainer] 和 xref:vc/common-attributes.adoc#property[property] xml 属性。xref:vc/components/entityComboBox.adoc#itemsContainer[itemsContainer] 属性用于创建选项列表：

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/multiselectcombobox/multi-select-combo-box-view.xml[tags=data;userDc;hobbiesDc;layout;basics]
----

<1> `User` 实体的 `InstanceContainer`。
<2> 数据容器内部的实体fetch plan。
<3> `Hobby` 实体的 `CollectionContainer`。
<4> `multiSelectComboBox` 使用 `hobbiesDc` 作为选项列表数据容器。

组件返回选择项目的列表。
